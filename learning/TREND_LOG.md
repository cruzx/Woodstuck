# 趋势蒸馏记录

这个文件是联网趋势学习的入口索引，不再承载所有完整记录。

完整趋势条目按月份存放在 `learning/trends/`，避免单个日志越来越大。

只记“值得复用的判断”，不记大段新闻摘要。

## 写入规则

- 新增联网学习记录时，写入当月文件：`learning/trends/YYYY-MM.md`
- `TREND_LOG.md` 只维护索引、格式和读取规则
- 如果某条信号跨任务稳定，再升级到 `PROFILE.md`
- 不要把短期热词、链接列表、无迁移价值的截图说明写进月度记录

## 记录格式

复制下面结构追加到当月文件：

```md
## YYYY-MM-DD

- 主题：
- 外部信号：
- 为什么值得在意：
- 对当前任务的可迁移点：
- 不能直接照抄的风险：
- 是否升级到 PROFILE：
```

## 使用原则

- 优先记录跨任务有价值的机制变化
- 少记纯视觉流行，多记交互、商业、创作效率、用户心智变化
- 每条尽量能回答：`这会让我们以后判断得更准吗`
- 近期趋势判断优先读当前月份；需要追溯时再读历史月份
